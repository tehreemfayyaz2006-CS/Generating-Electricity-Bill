⚡ Electricity Bill Generator
📌 Project Description
This project is a simple Electricity Bill Generator System that calculates the total electricity bill based on units consumed. It helps users quickly estimate their bill using a basic calculation formula.
🚀 Features
Input units consumed
Automatic bill calculation
Simple and user-friendly interface
Fast and accurate results
Beginner-friendly project
🛠️ Technologies Used
HTML (structure)
CSS (styling)
JavaScript (logic)
